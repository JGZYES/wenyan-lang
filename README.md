# Flarum 文言语言包

[![GitHub license](https://img.shields.io/github/license/JGZYES/wenyan-lang?color=%230172b2)](https://github.com/JGZYES/wenyan-lang/blob/master/LICENSE) [![Latest Stable Version](https://img.shields.io/packagist/v/JGZYES/wenyan-lang.svg)](https://packagist.org/packages/JGZYES/wenyan-lang) ![Packagist](https://img.shields.io/packagist/dt/JGZYES/wenyan-lang)

是语言包也，本于 [flarum/lang-english](https://github.com/flarum/lang-english)、[jsthon 2015](https://discuss.flarum.org/d/612) 及 [Csineneo 2019](https://github.com/Csineneo/lang-simplified-chinese) 诸本。

文风细腻自然，合华人思维之常。

欲览简体，请往此处：[简体中文语言包](https://discuss.flarum.org/d/22690)。

### 所需 / Require
  - Flarum v1.8.0 及以上版本

### 安装 / Install

**_Flarum 最新版_**
```bash
composer require jgzyes/wenyan-lang
php flarum cache:clear
```
